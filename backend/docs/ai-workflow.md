# AI Workflow

## Data Flow

Employee Data
    ↓
Performance Score Analysis
    ↓
Recommendation Engine
    ↓
Employee Classification
    ↓
Dashboard Analytics

## Recommendation Logic

Score >= 85
→ Top Performer
→ Promotion + Leadership Program

Score 60–84
→ Average Performer
→ Skill Enhancement Training

Score < 60
→ Needs Improvement
→ Technical Mentoring Required
